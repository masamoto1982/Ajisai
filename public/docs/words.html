Target Specification

.

Stack top mode (default)

[ 1 2 3 ] LENGTH # Operates on top

..

Stack mode - operates on entire stack

a b c .. REVERSE # Result: c b a

Position Operations (0-indexed)

GET

Get element at position. Negative index from end.

[ 10 20 30 ] [ 0 ] GET # Result: [ 10 ] [ 10 20 30 ] [ -1 ] GET # Result: [ 30 ]

INSERT

Insert element at position

[ 1 3 ] [ 1 ] [ 2 ] INSERT # Result: [ 1 2 3 ]

REPLACE

Replace element at position

[ 1 2 3 ] [ 0 ] [ 9 ] REPLACE # Result: [ 9 2 3 ]

REMOVE

Remove element at position

[ 1 2 3 ] [ 1 ] REMOVE # Result: [ 1 3 ]

Vector Operations

LENGTH

Get element count

[ 1 2 3 4 5 ] LENGTH # Result: [ 5 ]

TAKE

Take N elements from start (positive) or end (negative)

[ 1 2 3 4 5 ] [ 3 ] TAKE # Result: [ 1 2 3 ] [ 1 2 3 4 5 ] [ -2 ] TAKE # Result: [ 4 5 ]

CONCAT

Concatenate vectors

[ a ] [ b ] CONCAT # Result: [ a b ]

REVERSE

Reverse order

[ 1 2 3 ] REVERSE # Result: [ 3 2 1 ]

RANGE

Generate range (inclusive)

[ 1 ] [ 5 ] RANGE # Result: [ 1 2 3 4 5 ]

SORT

Sort ascending

[ 3 1 4 1 5 ] SORT # Result: [ 1 1 3 4 5 ]

Tensor Operations

SHAPE

Get shape

[ [ 1 2 3 ] [ 4 5 6 ] ] SHAPE # Result: [ 2 3 ]

RESHAPE

Change shape

[ 1 2 3 4 5 6 ] [ 2 3 ] RESHAPE # Result: [ [ 1 2 3 ] [ 4 5 6 ] ]

TRANSPOSE

Transpose 2D

[ [ 1 2 ] [ 3 4 ] ] TRANSPOSE # Result: [ [ 1 3 ] [ 2 4 ] ]

FILL

Create filled tensor

[ 2 3 ] [ 0 ] FILL # Result: [ [ 0 0 0 ] [ 0 0 0 ] ]

Arithmetic

+ - * /

Element-wise with broadcasting

[ 1 2 3 ] [ 10 ] + # Result: [ 11 12 13 ] [ 1 2 3 ] [ 4 5 6 ] * # Result: [ 4 10 18 ]

MOD FLOOR CEIL ROUND

Math operations

[ 7 ] [ 3 ] MOD # Result: [ 1 ] [ 7/3 ] FLOOR # Result: [ 2 ]

Higher-Order Functions

MAP

Apply word to each element

[ 1 2 3 ] 'DOUBLE' MAP # Result: [ 2 4 6 ]

FILTER

Keep matching elements

[ 1 2 3 4 ] 'EVEN?' FILTER # Result: [ 2 4 ]

FOLD

Reduce with initial value

[ 1 2 3 4 ] [ 0 ] '+' FOLD # Result: [ 10 ]

Word Management

DEF

Define word

[ '[ 2 ] *' ] 'DOUBLE' DEF

DEL

Delete word

'DOUBLE' DEL

?

Show definition

'DOUBLE' ?

Control Flow

TIMES

Repeat N times

'DOUBLE' [ 3 ] TIMES

WAIT

Execute after delay (ms)

'PROCESS' [ 1000 ] WAIT
