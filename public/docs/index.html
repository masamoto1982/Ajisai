Playground

はじめに

Ajisai は AI-first・ベクトル指向・分数データフローのスタック言語です。このリファレンスでは、各機能を短い用例とともに紹介します。

すべての用例には 「Playground で開く」 ボタンが付いています。押すとアプリ本体のエディタにコードが流し込まれるので、その場で実行して挙動を確かめられます。

※ これはラフ版です。内容は段階的に充実させていきます。

値とベクトル

Ajisai のあらゆる値はベクトルに包まれます。角括弧 [ ] で囲み、要素は空白で区切ります。

[ 1 2 3 ]
Playground で開く

数値は分数（有理数）として扱えます。

[ 7/3 ]
Playground で開く

演算

四則演算はベクトル同士の要素ごとに作用します。

[ 1 2 3 ] [ 10 20 30 ] +
Playground で開く

Result（例）

[ 11 22 33 ]

剰余 MOD

[ 7 ] [ 3 ] MOD PRINT
Playground で開く

Result（例）

[ 1 ]

出力

PRINT はスタック最上段の値を出力エリアに表示します。

[ 42 ] PRINT
Playground で開く

範囲生成

RANGE は開始値と終了値から連続したベクトルを作ります。

[ 0 ] [ 5 ] RANGE PRINT
Playground で開く

Result（例）

[ 0 1 2 3 4 5 ]

Word の定義

コード片に名前を付けて新しい Word を定義できます。定義は DEF で行います。

[ '[ 2 ] *' ] 'DOUBLE' DEF
[ 21 ] DOUBLE PRINT
Playground で開く

Result（例）

[ 42 ]
