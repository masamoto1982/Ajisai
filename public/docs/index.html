Welcome to Ajisai

What is Ajisai?

Ajisai is a stack-based programming language inspired by FORTH, featuring a WebAssembly-based interpreter and an interactive web-based GUI.

The name "Ajisai" (hydrangea in Japanese) represents how small words come together to form functionality, like how small flowers form a hydrangea cluster.

📚

Stack-based RPN

FORTH-style stack operations with Reverse Polish Notation

🧮

Vector Operations

NumPy/APL-style broadcasting and tensor operations

🔢

Exact Fractions

All numbers as fractions - no rounding errors

🚀

WebAssembly

Rust core compiled to WASM for fast browser execution

Quick Example

# Vector creation and operations
[ 1 2 3 ]               # 1D vector
[ [ 1 2 ] [ 3 4 ] ]     # 2D nested vector

# Broadcasting arithmetic
[ 5 ] [ 1 2 3 ] +       # Result: [ 6 7 8 ]

# Custom word definition
[ '[ 2 ] *' ] 'DOUBLE' DEF
[ 5 ] DOUBLE            # Result: [ 10 ]

# Higher-order functions
[ 1 2 3 4 5 ] 'DOUBLE' MAP    # Result: [ 2 4 6 8 10 ]
