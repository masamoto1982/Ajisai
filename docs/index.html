What is Ajisai?

Ajisai is a stack-based programming language inspired by FORTH, featuring a WebAssembly-based interpreter and an interactive web-based GUI. The name "Ajisai" (hydrangea in Japanese) represents how small words come together to form functionality, like how small flowers form a hydrangea cluster.

Key Features

📚

Stack-based RPN

FORTH-style stack operations with Reverse Polish Notation for intuitive data manipulation.

🧮

Vector Operations

Nestable vectors with NumPy/APL-style broadcasting and tensor operations (SHAPE, RESHAPE, TRANSPOSE).

🔢

Exact Fractions

All numbers are internally represented as fractions - no rounding errors, even for very large numbers.

🚀

WebAssembly

Core interpreter written in Rust, compiled to WebAssembly for fast execution in the browser.

Quick Example

# Vector creation and operations
[ 1 2 3 ]               # 1D vector: [1, 2, 3]
[ [ 1 2 ] [ 3 4 ] ]     # 2D nested vector (matrix-like)

# Broadcasting arithmetic
[ 5 ] [ 1 2 3 ] +       # Result: [ 6 7 8 ]

# Custom word definition
[ '[ 2 ] *' ] 'DOUBLE' DEF
[ 5 ] DOUBLE            # Result: [ 10 ]

# Higher-order functions
[ 1 2 3 4 5 ] 'DOUBLE' MAP    # Result: [ 2 4 6 8 10 ]
