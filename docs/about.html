AI-Driven Development

The majority of this project's implementation was done by AI (Claude). From design decisions to Rust/TypeScript code implementation, test case creation, and documentation, this project is developed through human-AI collaboration.

This represents an experimental approach to software development where:

  • Human provides high-level design decisions and direction
  • AI implements detailed code, tests, and documentation
  • Iterative refinement through human-AI dialogue

Technology Stack

Rust

Core interpreter implementation with memory safety and high performance.

WebAssembly

Browser runtime for near-native execution speed.

TypeScript

Type-safe frontend for interactive GUI.

Vite

Modern build tool for fast development.

GitHub Actions

CI/CD for automated builds and deployment.

wasm-pack

Rust to WebAssembly compilation toolchain.

Local Development

Prerequisites

  • Rust
  • wasm-pack
  • Node.js (v20 or later recommended)

Setup

# Clone the repository
git clone https://github.com/masamoto1982/Ajisai.git
cd Ajisai

# Install dependencies
npm install

# Build WASM
cd rust
wasm-pack build --target web --out-dir ../js/pkg
cd ..

# Build TypeScript
npm run build

# Start development server
npx vite

Production Build

npx vite build
